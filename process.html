Process

The Pitch

A cute, family friendly 2D Platformer

Switch between 3 characters: Baby Bear, Mama Bear, and Papa Bear Each bear has different properties, such as weight, speed, and jump height Carefully time how you switch to each bear and catch that no good Rolldilocks!

Replay each level to get a gold star time, and find collectibles!

Lo-Fi Prototypes

Level Select


Design Questions Asked
How does the player choose to move between scenes? How does the level selection scene make navigating the game easier? If there are different worlds how will the level selection scene incorporate this into scene selection? How do we show progress with each level in this prototype?

Knowledge Gained
We learned how to create a button object that can be used to navigate to other scenes. Hierarchy matters when dealing with multiple objects in Unity. We learned how to use assets and import them into button objects. We started exploring different icons for the levels

Splash Animation

Design Questions Asked
What sequence of screens will the player progress through to reach the first moment of core gameplay? Which special animations or transitions can be used to add a high level of design polish to these first few seconds?
We wanted to create a way to let the user know the game is finished loading in. The animation will play when the game has loaded everything in. After that, it will lead them to a world/level selection, allowing the user to see what they have accomplished and what worlds/levels they can play.


Hi-Fi Prototypes

Scene Flow


Design Questions Asked
What is our scene hierarchy, what's the order of flow? What will the user's experience be between starting the game and playing the game? Which scenes are relevant to the scene flow? Is the flow intuitive for the player?
We learned how to use a cohesive button design to flow from scene to scene. We wanted to keep the experience short with a skippable splash animation and display information on the level select screen instead of navigating through a screen.
